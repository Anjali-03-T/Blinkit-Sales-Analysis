# 🛒 Blinkit Sales Analysis Dashboard

## 📌 Project Overview

This project analyzes Blinkit sales data using **Excel, Python, MySQL, and Power BI**. The goal is to clean and prepare retail sales data, perform SQL-based analysis, and build an interactive dashboard to uncover business insights related to sales performance, customer behavior, and product trends.

---

## 🎯 Objectives

- Clean and preprocess raw sales data.
- Import cleaned data into MySQL.
- Perform SQL queries for business analysis.
- Build an interactive Power BI dashboard.
- Generate actionable insights for decision-making.

---

  - Delivery Type
  - Payment Mode

---

## 💡 Business Insights

- Dairy and Grocery categories generated the highest revenue.
- Delhi recorded the highest overall sales.
- UPI was the most preferred payment method.
- Customer ratings remained consistently high across sellers.
- Monthly sales showed fluctuations, highlighting seasonal demand patterns.
